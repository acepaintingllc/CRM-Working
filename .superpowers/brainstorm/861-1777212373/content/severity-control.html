A — Segmented buttons (None / Minor / Moderate / Major)

Binary modifiers are checkboxes. Severity modifiers show a 4-level segmented button row. "None" means off. Fast to tap, clearly shows current level, easy to change.

B — Checkbox + slider for severity

Check to enable. If severity applies, a slider appears inline. More granular control but requires more precision to set. Slider snaps to predefined stops internally.

C — Dropdown per modifier

All modifiers use a consistent dropdown: "— none —" means off, otherwise pick a level. No separate checkbox needed. Slightly more visual weight but very uniform — binary modifiers just have fewer options.
